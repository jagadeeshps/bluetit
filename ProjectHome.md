# Bluetit #
## Open source web base application to manage project agility by scrum. ##

---



### Features ###
  * Project Backlog
  * Project Progress Trent
  * Sprint Backlog
  * Burn-Down chart
  * User Story Planning
  * Estimate Poker
  * Sprint Team Time-sheet
  * Wiki Style documentation for requirement
  * Discussion forum

### Other Features ###
  * Role base Dashboard
  * Customization Dashboard
  * Plugins
